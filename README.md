# Personal Finance Management App

## Undergraduate Thesis: Information Technology Engineer

### Project Overview
This application was developed as the centerpiece of my thesis, focusing on helping individuals plan, monitor, and control their finances. [cite_start]It provides a simple and efficient way to gain insight into financial status through a modern user interface[cite: 56, 58].

### Key Features
* [cite_start]**User Authentication**: Secure login and registration system for personal data protection.
* [cite_start]**Transaction Management**: Users can create, search, filter, and delete income and expense records.
* [cite_start]**Data Visualization**: Financial flows are represented through different types of interactive charts for better analysis.
* [cite_start]**Client-side Persistence**: Utilization of **LocalStorage** to ensure data remains available across sessions without an external database[cite: 57, 58].

### Technical Stack
* [cite_start]**Frontend**: React (Functional Components, Hooks), JavaScript (ES6+)[cite: 66, 67].
* [cite_start]**Styling**: **CSS Modules** for modular and conflict-free styling.
* [cite_start]**Build Tool**: **Vite** for optimized development performance.
* [cite_start]**Version Control**: **Git & GitHub**[cite: 70].
